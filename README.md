# Vue

## vue cli 3.0 创建项目
```
打开命令行 vue ui 

( 以图形化界面创建和管理项目 )

配置一般选择：vue-router、dart-scss、babel、eslint
```

## npm run serve 默认打开浏览器
```
package.json 找到下面一行代码

"serve": "vue-cli-service serve"

修改为

"serve": "vue-cli-service serve --open"
```
